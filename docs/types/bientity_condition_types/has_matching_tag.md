#   Has Matching Tag

[**Bi-entity Condition Type**][1]

Checks whether the actor entity and the target entity has matching scoreboard tag(s) *(added via the `/tag` command).*

Type ID: `eggolib:has_matching_tag`


### Fields

*None.*


### Examples

=== "Example #1"

    ``` json
    "bientity_condition": {
        "type": "eggolib:has_matching_tag"
    }
    ```
