#   Block Item

[**Item Condition Type**](../item_condition_types.md)

Checks whether the item is a block item *(e.g: an item that can place blocks).*

Type ID: `eggolib:block_item`


### Fields

None.


### Examples

``` json
"item_condition": {
    "type": "eggolib:block_item"
}
```
