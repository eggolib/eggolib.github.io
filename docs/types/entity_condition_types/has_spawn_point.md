#   Has Spawn Point

[**Entity Condition Type**][1]

Checks if the entity has a spawnpoint.

Type ID: `eggolib:has_spawn_point`


!!! note

    This entity condition type will only work for players.


### Fields

*None.*


### Examples

=== "Example #1"

    ```json
    "condition": {
        "type": "eggolib:has_spawn_point"
    }
    ```
