#   Crawling

[**Entity Condition Type**][1]

Checks if the entity that has the power is crawling (e.g: in swimming pose and not submerged in water).

Type ID: `eggolib:crawling`


### Fields

_None._


### Examples

```json
"condition": {
    "type": "eggolib:crawling"
}
```
