#   Moon Phase

[**Data Type**][1]

A [**string**][2] used to determine the moon phase of a world.


!!! note

    See [**Minecraft Fandom Wiki: Moon (Phases)**][3] for more information about moon phases.


### Values

Value | Index
------|------
`"full_moon"` | `0`
`"waning_gibbus"` | `1`
`"last_quarter"` | `2`
`"waning_crescent"` | `3`
`"new_moon"` | `4`
`"waxing_crescent"` | `5`
`"first_quarter"` | `6`
`"new_moon"` | `7`
