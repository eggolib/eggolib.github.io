#   Render Mode

[**Data Type**][1]

A [**string**][2] used to determine how a component is rendered.


### Values

Value | Description
------|------------
`default` | Renders the component as is.
`hide_partially` | Renders the component semi-transparently.
`hide_completely` | Does not render the component at all.
