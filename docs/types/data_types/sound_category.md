#   Sound Category

[**Data Type**][1]

A [**string**][2] used for grouping sound events together.


### Values

* `ambient`
* `blocks`
* `hostile`
* `master`
* `music`
* `neutral`
* `players`
* `records`
* `voice`
* `weather`
